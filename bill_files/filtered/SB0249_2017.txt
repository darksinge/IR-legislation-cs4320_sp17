Bramble      House Sponsor   Daniel  McCay             LONG TITLE      General Description            This bill modifies the format that is required for filing certain income tax withholding       statements.       Highlighted Provisions             This bill                    requires a producer to file a quarterly mineral production withholding return in an       electronic format  and                   requires an employer to file a quarterly withholding return in an electronic format.       Money Appropriated in this Bill             None       Other Special Clauses             This bill provides a special effective date.
