 REPEAL OF 1933 UTAH CODE REVISION STATUTES



            

    
2001 GENERAL SESSION

            

    
STATE OF UTAH

            

    


 Sponsor:  Millie M. Peterson

            

    

 This act modifies the Statutes Title.  This act repeals certain obsolete and lapsed provisions
            

    
 relating to the 1933 recodification of the Utah code.
            

    
This act affects sections of Utah Code Annotated 1953 as follows:
            

    
REPEALS:
            

    
     68-2-1, Utah Code Annotated 1953
            

    
     68-2-3, Utah Code Annotated 1953
            

    
     68-2-4, Utah Code Annotated 1953
            

    
     68-2-5, Utah Code Annotated 1953
            

    
     68-2-6, Utah Code Annotated 1953
            

    
     68-2-7, Utah Code Annotated 1953
            

    
     68-2-8, Utah Code Annotated 1953
            

    
     68-2-9, Utah Code Annotated 1953
            

    
     68-2-10, Utah Code Annotated 1953
            

    
Be it enacted by the Legislature of the state of Utah:
            

    
    Section 1.  Repealer.
            

    
    This act repeals:
            

    
    Section  68-2-1,  Short title  Effective date.
            

    
    Section  68-2-3,  General repeal -- Of prior laws -- Exceptions and reservations.
            

    
    Section  68-2-4,  Laws of 1933 deemed subsequent to revision.
            

    
    Section  68-2-5,  Tenure of office not affected -- Exceptions.
            

    
    Section  68-2-6,  Accrued rights not affected by repeal.
            

    
    Section  68-2-7,  Effect on limitation of actions.
            

    
    Section  68-2-8,  Effect on offenses committed.
            

    
    Section  68-2-9,  Effect on suits and prosecutions pending.
            

    
    Section  68-2-10,"Heretofore" and "hereafter" defined.
